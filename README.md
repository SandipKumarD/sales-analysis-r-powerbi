# 📊 Sales Analysis Project (R + Power BI)

## 🔍 Overview
This project analyzes synthetic beverage sales data using R for data processing and Power BI for visualization.

## 🛠 Tools Used
- R (dplyr, data analysis)
- Power BI (dashboard, DAX)

## 📊 KPIs
- Total Revenue
- Total Orders
- Total Customers
- Average Order Value
- Product & Category Analysis

## 📈 Dashboard Features
- Revenue by Category & Region
- Top & Least Products
- Customer Segmentation (B2B vs B2C)
- Time-series Analysis

## 💡 Insights
- Top products drive majority of revenue
- B2C customers dominate sales
- Seasonal trends observed in revenue

## 📂 Files
- analysis.R → R code
- dataset.csv → data
- dashboard.pbix → Power BI dashboard

## 🚀 Conclusion
This project demonstrates end-to-end data analysis from data processing to visualization.
